# Cuanstrat — Tools Bantu Analisa Saham IDX

Terima kasih sudah mengunduh Cuanstrat!

## ⚠️ Disclaimer

Aplikasi ini hanyalah **tools bantu analisa teknikal**, BUKAN ajakan
atau rekomendasi jual/beli saham. Segala keputusan transaksi adalah
risiko dan tanggung jawab Anda sendiri. Selalu lakukan riset tambahan
sebelum mengambil keputusan trading.

## Cara Install

1. Pindahkan file `app-release.apk` ke HP Android Anda (lewat kabel
   USB, Google Drive, email ke diri sendiri, atau cara lain).
2. Buka file `.apk` itu di HP untuk mulai instalasi.
3. Kalau muncul peringatan "Install dari sumber tidak dikenal" atau
   "Untuk keamanan Anda...", ini **wajar** karena APK ini tidak
   diunduh dari Google Play Store. Ikuti petunjuk di HP untuk
   mengizinkan instalasi tetap berjalan.
4. Setelah selesai, buka aplikasi "Cuanstrat" dari menu HP Anda.

## Cara Pakai

- **7 tombol strategi** (BSJP, Bandar Breakout, Bandar Spike, Bandar
  Akumulasi, Gorengan Hunter, Big Money Flow, Akumulasi Sesi 2): tekan
  salah satu untuk men-screening seluruh saham IDX berdasarkan
  kriteria strategi tersebut. Proses ini butuh waktu beberapa menit
  karena mengecek ratusan saham satu per satu — mohon ditunggu sampai
  selesai.
- **Ikon gear (⚙️) di samping tiap tombol strategi**: untuk mengatur
  ulang angka kriteria (threshold) strategi tersebut sesuai preferensi
  Anda. Ada tombol "Default" untuk kembalikan ke nilai semula.
- **Kolom "Cek Satu Saham"**: ketik kode saham (misal `BBRI`), tekan
  "Cari" untuk mengecek 1 saham itu dengan ketujuh strategi sekaligus.
- **Hasil screening**: akan muncul dalam bentuk pop-up berisi teks,
  dengan tombol **"Salin ke Clipboard"** untuk menyalin hasilnya
  (misal untuk ditempel ke catatan, WhatsApp, dll).
- **Ikon gear (⚙️) di pojok kanan atas**: membuka halaman Settings,
  tempat Anda bisa mengisi API Key Gemini milik Anda sendiri (opsional)
  untuk mengaktifkan analisa AI tambahan saat mencari 1 saham. Kalau
  tidak diisi, aplikasi tetap berjalan normal hanya dengan analisa
  matematis — tanpa biaya tambahan apapun.

## Tentang Analisa AI (Opsional)

Aplikasi ini mendukung analisa AI tambahan menggunakan Gemini API,
dengan sistem **BYOK (Bring Your Own Key)** — artinya Anda perlu
API Key Gemini milik Anda sendiri untuk mengaktifkan fitur ini.

- Dapatkan API Key gratis di [Google AI Studio](https://aistudio.google.com/apikey)
- Masukkan API Key tersebut di halaman Settings aplikasi
- Biaya pemakaian API (jika ada) sepenuhnya tanggung jawab Anda sebagai
  pemilik API Key — developer aplikasi ini tidak menanggung biaya
  apapun dari penggunaan fitur ini

## Update Aplikasi

Aplikasi ini akan otomatis mengecek versi terbaru setiap kali dibuka.
Jika ada versi baru, akan muncul notifikasi dengan tautan ke halaman
unduhan. Karena aplikasi ini didistribusikan di luar Play Store, Anda
perlu mengunduh dan menginstall ulang APK terbaru secara manual setiap
kali ada update.

## Kritik, Saran, atau Menemukan Bug?

Di dalam aplikasi, tekan tombol back/keluar di halaman utama untuk
membuka tombol **"Kritik & Saran"** yang akan mengarahkan Anda ke
aplikasi Email.

## Dukung Pengembangan

Aplikasi ini gratis dan dikembangkan secara mandiri. Jika bermanfaat,
Anda bisa memberikan dukungan lewat QRIS yang muncul di pop-up saat
Anda menutup aplikasi. Dukungan apapun sangat dihargai 🙏
